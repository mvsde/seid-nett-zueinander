Seid nett zueinander

Badge for the promotion “Seid nett zueinander” of the German newspaper “Hamburger Abendblatt”.

Usage

Use the jsDelivr CDN to easily include the badge:

<script src="https://cdn.jsdelivr.net/npm/seid-nett-zueinander" defer></script>
